Grid Container (hasResizableColumns="true", withHandles="true", handleClasses="dijitTitlePaneTitle", isOffset="true")

  • properties : withHandles="true" and handleClasses="dijitTitlePaneTitle" enable the DnD only on the dijitTitlePaneTitle.
  • the property : acceptTypes="dijit.layout.ContentPane, dijit.TitlePane" enables the Dnd for ContentPane and TitlePane. You can not DnD for example the calendar or ColorPalette widgets.
