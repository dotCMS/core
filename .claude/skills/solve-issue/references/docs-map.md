# Docs Map

Load these with the `Read` tool before writing any code in Step 7b.
Docs are in the repo root `docs/` directory — use paths relative to the repository root.
If a file does not exist, skip it and continue.

---

## Always load

- `core-web/CLAUDE.md` — Angular project conventions (overrides general standards)

## Load by change type

| Doc | When |
|-----|------|
| `docs/frontend/ANGULAR_STANDARDS.md` | Any `.ts` or `.html` change |
| `docs/frontend/COMPONENT_ARCHITECTURE.md` | Creating or modifying a component |
| `docs/frontend/STYLING_STANDARDS.md` | Any `.scss` or HTML template change |
| `docs/frontend/TESTING_FRONTEND.md` | Creating or updating `.spec.ts` files |
| `docs/frontend/STATE_MANAGEMENT.md` | Change involves NgRx or Signal Store |
